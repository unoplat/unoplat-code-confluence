# Business Domain References

## Domain Summary
This project is an OpenMetadata ingestion connector for Code Confluence, transforming repository-analysis snapshots into cataloged developer-tooling assets. It models codebases with language, engineering workflow, dependency, business-logic, core-file evidence, and application-interface metadata, then publishes repository API services, codebase collections, and detected HTTP endpoints. Its dominant domain is evidence-backed source-code context and discovery for developers and AI coding agents.

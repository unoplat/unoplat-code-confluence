# Business Domain References

## Domain Summary
This repository is an OpenMetadata ingestion connector for Code Confluence, a repository-analysis system that turns query-engine snapshots into catalog metadata. Its main concern is modeling each repository and codebase with programming language, engineering workflow, dependency, business-logic, and interface information so that the results can be published and searched in OpenMetadata. Overall, the business domain is deterministic source-code understanding and discovery for developer tooling and metadata cataloging.

# Business Domain References

## Domain Summary
This project is an OpenMetadata ingestion connector for Code Confluence, transforming repository-analysis snapshots into catalog assets. It models codebases’ engineering workflows, dependencies, business-logic evidence, and application interfaces, publishing them as API services, collections, and HTTP endpoints. Its core domain is developer-tooling metadata that makes repository behavior and source-code context discoverable for developers and AI agents.

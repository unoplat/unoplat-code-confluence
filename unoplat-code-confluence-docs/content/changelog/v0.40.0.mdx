## Highlights

This release scaffolds the new `unoplat-code-confluence-cli`, giving the project a first-class command-line entry point for running the stack. The CLI uses python-on-whales for its Compose runtime, models its runtime state with pydantic, and is wired into the build and release pipelines.

## What's New

### Command-Line Interface

- Scaffolded the initial `unoplat-code-confluence-cli`
- Used python-on-whales for the Compose runtime
- Migrated CLI runtime models from dataclasses to pydantic
- Added a dev dependency group with ruff and pyrefly
- Added the CLI package to the build and release pipelines

## Bug Fixes

- Removed an unnecessary docker compose check and made the CLI release step reliable

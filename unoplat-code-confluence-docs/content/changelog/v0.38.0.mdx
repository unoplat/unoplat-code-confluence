## Highlights

This release replaces structural-signature extraction with `BaseSourceContext` detection primitives in the ingestion pipeline and routes query-engine output by workflow status, prefetching the prior run to decide how results are dispatched.

## What's New

### Detection Primitives

- Replaced structural-signature extraction with `BaseSourceContext` detection primitives in ingestion

### Status-Based Output Dispatch

- Pre-fetched the previous workflow and used status-based output dispatch in the query engine

## Bug Fixes

- Made `GitHubUser` fields optional and handled null display values in the frontend
- Fixed a logging issue

# Business Logic References

## Content Metadata Models

- `src/lib/banner-config.ts`
  - Defines the `BannerConfig` data used to render marketing or alert banners, including identifiers, message content, optional link metadata, and visual variants.
- `src/lib/changelog-utils.ts`
  - Defines the `ChangelogEntry` metadata for release notes, capturing versioning, dates, summaries, and optional GitHub release linkage.
- `src/lib/seo.ts`
  - Defines SEO metadata options for documentation pages, including title, description, path, and page type.

# Business Domain References

## Domain Summary
This codebase serves a code-confluence platform for ingesting Git repositories and codebases, then extracting structural signatures from Python and TypeScript source files. It models framework feature catalogs and detections, repository/codebase metadata, workflow runs and agent event streams, plus supporting concerns like credentials, flags, language/package-manager metadata, and PR publication tracking. Overall, the dominant business domain is automated code analysis and repository intelligence for generating and managing structured documentation or agent output around codebases.

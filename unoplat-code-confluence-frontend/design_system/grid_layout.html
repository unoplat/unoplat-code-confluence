Component Grid Patterns

Card Grid Layout

Card grids use a 24px gap (gap-6) with responsive columns that adjust based on screen width. For best readability, limit to 3 columns on large screens.

Card List Layout

List layouts stack items vertically with 16px spacing (space-y-4) between items. Items use a horizontal flex layout for responsive content alignment.

Modal & Dialog Positioning

Modal Centering

Modals should be fixed position (fixed inset-0) with z-index 50 or higher (z-50). Center content with flex and limit width based on content type (max-w-md for small, max-w-2xl for medium, max-w-4xl for large).

Notification Placement

Toast Notification Positioning

Toast notifications should be positioned at the bottom center (bottom-4 inset-x-0) with a z-index of 40+. Stack multiple notifications vertically with 16px spacing (space-y-4) and center them horizontally.
